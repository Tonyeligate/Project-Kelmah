# Ticket - Carpentry Cabinet Making

**Date**: March 19, 2026

## Summary
Turn cabinet making into a distinct carpentry ticket with measurement, finish, and install metadata.

## Job Tags
- Jobs: `Jobs`, `Search`, `Quick Jobs`, `Payments`
- Skill Tags: `Carpentry`, `Cabinet Making`, `Woodwork`, `Installation`

## Product Gaps
- Cabinet work is hidden under generic carpentry.
- Users need dimension and finish input.
- Workshop vs mobile service is not clearly labeled.

## Implementation Focus
1. Add cabinet making as a distinct carpentry subtrade.
2. Add dimension, wood type, and finish fields.
3. Add workshop/mobile service tags.
4. Add before/after gallery and quotation support.

## Acceptance Notes
- Cabinet making should be searchable by kitchen, wardrobe, and custom furniture terms.
- The request form should support measurements without forcing long text entry.
