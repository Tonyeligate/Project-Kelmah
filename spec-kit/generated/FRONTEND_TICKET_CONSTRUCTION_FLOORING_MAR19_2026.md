# Ticket - Flooring Installation

**Date**: March 19, 2026

## Summary
Turn flooring into a distinct job type with material and area measurements.

## Job Tags
- Jobs: `Jobs`, `Search`, `Quick Jobs`, `Payments`
- Skill Tags: `Flooring`, `Tiling`, `Construction`, `Installation`

## Product Gaps
- Flooring is currently too broad and not measurement-friendly.
- Users need floor type and square-meter input.
- Install vs repair vs replacement is not separated clearly.

## Implementation Focus
1. Add flooring as a distinct search category.
2. Add floor type, area size, and installation/replacement fields.
3. Add material estimate and finish level options.
4. Add gallery support for completed work.

## Acceptance Notes
- Flooring should work well for both small rooms and whole-house jobs.
- The form should avoid long free-text requests when measurement fields can be used.
