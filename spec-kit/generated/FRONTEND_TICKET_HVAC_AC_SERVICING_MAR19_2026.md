# Ticket - HVAC AC Servicing

**Date**: March 19, 2026

## Summary
Split AC servicing into a direct request flow with cooling urgency and maintenance metadata.

## Job Tags
- Jobs: `Quick Jobs`, `Jobs`, `Search`, `Payments`
- Skill Tags: `HVAC`, `Air Conditioning`, `Servicing`, `Cooling`

## Product Gaps
- AC servicing is not clearly separated from installation.
- Cooling urgency and maintenance cycles are not visible enough.
- Users need refrigerant and system type info.

## Implementation Focus
1. Add AC servicing as a dedicated HVAC ticket.
2. Add service interval and refrigerant fields.
3. Add system type, residential/commercial, and urgency tags.
4. Add parts checklist and service record support.

## Acceptance Notes
- Servicing should appear as a recurring maintenance option.
- Mobile users should be able to book maintenance without complex forms.
