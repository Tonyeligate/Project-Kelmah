# Ticket - Plumbing Pump Installation

**Date**: March 19, 2026

## Summary
Create a dedicated plumbing ticket for pump installation and servicing, including water tanks and pressure systems.

## Job Tags
- Jobs: `Jobs`, `Quick Jobs`, `Worker Search`, `Payments`
- Skill Tags: `Plumbing`, `Pump Installation`, `Water Tank`, `Water Pressure`

## Product Gaps
- Pump installation is only implicit in generic plumbing.
- There is no field for tank type, pressure issue, or power source.
- Homeowners need a simpler way to quote material costs.

## Implementation Focus
1. Add pump installation as a searchable subtrade.
2. Add tank type, pressure issue, and power source fields.
3. Add material estimate and warranty note inputs.
4. Add before/after photo support for servicing jobs.

## Acceptance Notes
- Users should understand whether the job is install, repair, or upgrade.
- Search should return pump jobs when users type water tank or pressure issue.
