# Ticket - Maintenance Emergency Callout

**Date**: March 19, 2026

## Summary
Create a generic emergency callout ticket for handyman and maintenance repair jobs.

## Job Tags
- Jobs: `Quick Jobs`, `Jobs`, `Messaging`, `Payments`
- Skill Tags: `Maintenance`, `Handyman`, `Emergency`, `Repair`

## Product Gaps
- Emergency callout handling is not standardized.
- Users need a parts budget, urgency, and location priority.
- One-off repair vs recurring maintenance is not distinguished enough.

## Implementation Focus
1. Add emergency callout as a universal maintenance job type.
2. Add urgency, location, and response-time fields.
3. Add parts budget, checklist, and photo support.
4. Add one-off vs recurring service tags.

## Acceptance Notes
- Emergency callout should be the fastest path for small urgent repairs.
- Mobile booking must minimize typing and maximize clarity.
