# Ticket - Electrical Switchboard Upgrade

**Date**: March 19, 2026

## Summary
Capture switchboard upgrades as a dedicated electrical ticket with stronger safety and load metadata.

## Job Tags
- Jobs: `Jobs`, `Quick Jobs`, `Hirer Dashboard`, `Worker Search`
- Skill Tags: `Electrical`, `Switchboard`, `Breaker`, `Load Assessment`

## Product Gaps
- Switchboard work is not a distinct request type.
- Users need load details, breaker count, and photo evidence.
- Commercial vs residential context is missing.

## Implementation Focus
1. Add switchboard upgrade as a searchable subtrade.
2. Add breaker count and load assessment fields.
3. Add photo upload for the current panel.
4. Add commercial/residential and repair/new-install tags.

## Acceptance Notes
- Switchboard upgrades should be easy to distinguish from lighting and socket jobs.
- Safety messaging must appear before the user submits the request.
